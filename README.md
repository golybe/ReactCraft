# ReactCraft

Minecraft-like game built with React and Three.js.

## Features

- 3D voxel world generation
- Block breaking and placing
- Inventory system
- Physics simulation
- Multiplayer support (server-based)

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

This will start both the development server and the game server concurrently.

## Build

```bash
npm run build
```

## Technologies

- React
- Three.js
- React Three Fiber
- Vite
- Express
